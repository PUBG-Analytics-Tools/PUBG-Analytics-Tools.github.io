## 📌 About the Project

A collection of advanced performance analysis and telemetry tools for PUBG. Each tool provides real-time tracking, visual overlays, and gameplay improvement features.

---

## 🔧 Available Tools

- **FECURITY** — Secure and professional performance analyzer with precise aimbot, comprehensive ESP for players, vehicles, loot, and crates.
- **ARCANE** — Premium performance analyzer with highly accurate aimbot, advanced ESP for players, vehicles, loot, and crates through walls.
- **ARES FULL** — Complete performance analyzer with powerful aimbot, full-featured ESP for players, vehicles, loot, and supply drops.
- **ARES RADAR** — Focused radar-style analyzer with clean 2D/3D radar, real-time player, vehicle, and loot positions, and optional aim assistance.
